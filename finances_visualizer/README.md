# Visualizador de Finanças

> Minhas finanças nunca foram tão bonitas

A modern financial data visualization application built with React, TypeScript, and Tailwind CSS. Import your bank CSV files (Nubank, Inter) and visualize your financial data with beautiful interactive charts.

## Tech Stack

- **React 18** - UI framework with hooks and concurrent features
- **Vite** - Fast build tool with hot module replacement
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling with dark theme
- **Vitest** - Fast unit testing framework

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Available Scripts

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm run test

# Run tests with UI
npm run test:ui

# Run tests with coverage
npm run test:coverage

# Run linter
npm run lint

# Fix linting issues
npm run lint:fix

# Type check
npm run typecheck

# Format code
npm run format
```

## Project Structure

```
src/
├── components/    # React components
├── lib/          # Utility functions and helpers
├── types/        # TypeScript type definitions
├── hooks/        # Custom React hooks
└── test/         # Test setup and utilities
```

## Features

- Import CSV files from Nubank and Inter banks
- Automatic transaction categorization
- Interactive charts and visualizations
- Dark mode interface
- Responsive design
